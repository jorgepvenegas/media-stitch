Key pain points with vanilla JS

  • All state in global variables — no reactivity, manual DOM updates everywhere
  • Event listener wiring happens once at load — no easy way to refresh handlers
  • Large flat JS file — hard to test, hard to navigate
  • CSS works fine, but component scoping gets messy as complexity grows
